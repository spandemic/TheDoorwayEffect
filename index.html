Help Lethe collect their items and escape "The Doorway Effect" to get to school!

This short, engaging memory game is a simple but fun experience that anyone can enjoy and you can learn to connect with our main character, Lethe. This game was created by Andy Eng, Ashley Lu, Siwen Tao, and Erick Tung. Source code can be found here.

(The doorway effect happens when you pass through different “doorways,” or sceneries, and your memory declines. More information can be found here.)
